# 计算

## 1.有一题考察了自相关系数的一些知识，

用到了这个公式:

$$\frac{\sum e_{t} e_{t-1}}{\sum e_{t}^{2}} \approx \frac{\sum e_{t} e_{t-1}}{\sqrt{\sum e_{t}^{2}} \sqrt{e_{t-1}^{2}}}=\hat{\rho}$
$\mathrm{DW} \approx 2(1-\hat{\rho})$

还是比较少见的，很多人在这上面卡壳。
